<Update label="June 17, 2026">

## New features

**Request new tag options while tagging uploads**

Uploaders in a public workspace portal can now request a new option from inside the tagging modal. When the option you need isn't in the list, type the value and submit it for approval. The request appears with an **Approval pending** badge so everyone knows it's still under review, and the upload can be saved without waiting.

Workspace admins see incoming requests in the field option request flow and can accept or decline them. Accepted options become available to all uploaders immediately.

**1-hour edit window for portal uploads**

Uploaders now have a 1-hour grace period after submitting an upload batch. During that window, you can adjust tags and details directly from the workspace uploads page. After the hour passes, the upload becomes read-only and an **Editing closed** notice explains why. The remaining time is shown as a hint while the window is still open.

**Organize shared galleries into sections**

The sharing gallery editor now lets you split a gallery into named sections—great for moments, days, or categories. Create, rename, reorder, and delete sections from the editor, choose between manual and dynamic media selection, and set the sort order per section. On the public gallery, viewers see each section as its own block with a clear divider when browsing all media.

**Smarter download profiles and crop presets**

Download profiles now expose clearer quality tiers and a refreshed crop preset picker so it's easier to pick the right output for each situation. The download modal also keeps your crop selection consistent across profiles, and warnings now appear up front when a profile or crop preset can't be applied to part of your selection.

## Improvements

- The tagging modal and tagging carousel now show clearer visuals for pending option requests, including a dedicated row when your search doesn't match an existing option.
- Search inside the tagging modal makes it easier to spot whether a value already exists or needs to be requested.
- The active upload panel and uploads page in the public portal surface the edit-window status so uploaders know exactly when changes are still possible.
- The public gallery navigation bar now highlights the active section, supports clearing search results in one click, and makes the "find myself" and "download all" actions easier to reach on mobile.
- Gallery settings expose banner display, find-myself, and download-action controls in a single, more consistent panel.

## Bug fixes

- Fixed media duplicating across sections when the same file appeared in more than one gallery.
- Fixed inconsistent sort order on the public gallery when switching between sections and the all-media view.
- Fixed the gallery layout occasionally rendering at the wrong height when sections were reordered.

</Update>
